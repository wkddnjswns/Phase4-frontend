### 🛠️ 개발 환경 설정 및 실행

프로젝트를 시작하고 실행할 수 있도록 환경을 설정하는 방법입니다.

1.  **필수 사항 확인:**
    * Node.js 
    * npm 또는 yarn

2.  **종속성 설치:**
    ```
    npm install
    또는
    yarn install
    ```

3.  **프로젝트 실행:**
    ```
    npm start
    또는
    yarn start
    ```
    * **실행 후:** 기본적으로 `http://localhost:3000` (포트는 환경에 따라 다를 수 있음)으로 사용자 사이트가 열립니다.

### 🔑 주요 접속 및 로그인 정보

#### 1. 사용자 사이트 

* **URL:** `http://localhost:[포트]`
* **로그인:** 이메일 주소, 비밀번호를 입력합니다.
    * **주의 사항:** 입력하는 이메일은 **유효한 이메일 형식**을 갖춰야 합니다. (ex : a@a) (실제 인증 과정은 생략됨)

#### 2. 관리자 사이트 

* **URL:** `http://localhost:[포트]/admin` (`/admin`을 붙여야 관리자 사이트로 접속)
* **접속 정보:**
    * **ID:** `admin`
    * **비밀번호:** `1234`
